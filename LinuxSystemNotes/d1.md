# What is Linux

Common Directories

```shell
/ #"root," the top of the file system hierarchy
/bin # binaries and other executable programs
/etc #system configuration files
/home #home directories
/opt #optional third party software
/tmp #temporary space, which is cleaned at boot file
/usr #user related programs
/dev #device files, typically controlled by the operating system and the system administrators
/var #variable data, most notable log files
```



```shell
/lib # system libraries
/lib64 # system libraries
/lost+found
/media
```

```shell
srv #server files
```
